Timestamp Microservice

Description:

Pass a Unix timestamp or natural language date as a parameter, and this service will return the date in both these forms.

Example:

https://tstamp-ms.herokuapp.com/December%2015,%202015

https://tstamp-ms.herokuapp.com/1450137600

Will give an output of:

{ "unix": 1450137600, "natural": "December 15, 2015" }
